## 🌟 项目简介

张大师是一款基于中国传统哲学「天人合一」理念的智能生活导航系统，将易经八卦、五行学说、黄历择吉等传统智慧与现代AI技术相结合，为现代都市人提供个性化的生活指导。

## ✨ 核心功能

### 🔮 智慧决策
- **用户画像**：基于生辰八字建立个人五行属性档案
- **今日运势**：实时运势分析与黄历宜忌提醒
- **择吉日历**：重要事项的吉日选择与规划

### 🍃 生活指南
- **衣**：根据节气五行推荐服饰搭配
- **食**：结合体质节气的养生食谱
- **住**：居家风水布局与作息建议
- **行**：出行方位与时辰选择

### 📚 文化传承
- **知识学习**：易经八卦、五行风水知识库
- **智慧问答**：传统文化疑问解答
- **社群共修**：分享交流传统智慧

## 🚀 快速开始

### 在线体验
直接打开 `index.html` 即可体验完整功能

### 本地运行
```bash
# 克隆项目
git clone https://github.com/joshua23/CaptainZhang.git

# 进入项目目录
cd CaptainZhang

# 直接打开index.html或使用本地服务器
# 使用Python
python -m http.server 8000

# 或使用Node.js
npx serve
```

## 🎨 设计特色

- **神秘优雅**：紫色系配色，营造仙风道骨氛围
- **动效丰富**：淡入淡出、光环脉冲、粒子飘动
- **响应式设计**：完美适配桌面、平板、手机
- **暗色模式**：符合中医养生作息规律

## 🛠️ 技术栈

- **前端**：HTML5 + CSS3 + JavaScript (原生)
- **设计**：响应式布局 + CSS动画
- **数据**：LocalStorage本地存储
- **算法**：黄历算法、五行推算、节气计算
